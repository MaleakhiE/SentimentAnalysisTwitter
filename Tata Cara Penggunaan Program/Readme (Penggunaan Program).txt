TATA - TATA CARA PENGGUNAAN PROGRAM


1. Lakukan Instalasi Python >= 3.9
2. Setelah atau sudah dilakukan instalasi Python, maka lakukan instalasi library
   yang digunakan dengan masuk kedalam terminal atau cmd dan lakukan perintah
   "pip install -r library.txt"
3. Jika sudah menginstall seluruh library yang diperlukan, tetap berada pada terminal
   atau cmd, dan masukan perintah "py -m streamlit run Program.py"
4. Program sudah dapat dijalankan.
